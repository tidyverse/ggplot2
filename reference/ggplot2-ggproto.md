# Base ggproto classes for ggplot2

If you are creating a new geom, stat, position, or scale in another
package, you'll need to extend from
[`ggplot2::Geom`](https://ggplot2.tidyverse.org/reference/Geom.md),
[`ggplot2::Stat`](https://ggplot2.tidyverse.org/reference/Stat.md),
[`ggplot2::Position`](https://ggplot2.tidyverse.org/reference/Position.md),
or [`ggplot2::Scale`](https://ggplot2.tidyverse.org/reference/Scale.md).
